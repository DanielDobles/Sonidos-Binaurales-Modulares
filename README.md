# Sonidos Binaurales Modulares (2026)

Una aplicación de síntesis de audio psicoacústico de alto rendimiento con una interfaz moderna, iridiscente y orientada a la experiencia de usuario.

## Características
- Síntesis de audio estéreo en tiempo real.
- Modulación rítmica de ondas portadoras.
- Visualización espectral dinámica.
- Estética glassmorphic moderna con Framer Motion.
